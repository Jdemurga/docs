The dashboard is the central view of Vexel Horizon, merging live market data with your personal or ecosystem-level financial intelligence. Every number on the screen updates in real time — there is no page refresh and no polling delay. Whether you are browsing as a guest or operating inside an authenticated Sovereign session, the dashboard surfaces the same layout with different data scoped to your context.

## Two modes

The dashboard operates in two distinct states depending on whether you have an active Sovereign Link session.

<Tabs>
  <Tab title="Collective Mode (guest)">
    When you visit Vexel Horizon without authenticating, the dashboard runs in **Collective Mode**. Instead of your personal wealth, you see aggregate data for the entire Vexel network.

    - The primary card displays **Global Ecosystem Value** — the total net worth across all Sovereign nodes on the Mesh.
    - The yield counter shows **Collective Velocity**, the ecosystem-wide capital growth rate per second.
    - The performance chart renders the **Vexel Ecosystem Index**, sourced from public history.

    Data is fetched from two public endpoints on load:

    ```
    GET /api/market/public/ecosystem
    GET /api/market/public/yield/status
    ```

    The feed also posts a `CORE` message showing the number of active Sovereign nodes currently linked to the Mesh.
  </Tab>
  <Tab title="Sovereign Mode (authenticated)">
    Once you complete a Sovereign Link session, the dashboard switches to **Sovereign Mode** and reloads with data scoped to your account.

    - The primary card label becomes **[Your Name] Vault Value**, populated from your stored profile.
    - The yield counter shows **Personal Velocity** — your individual capital growth rate per second.
    - The performance chart renders your **Personal Portfolio Performance**.

    Data is fetched from two authenticated endpoints on load:

    ```
    GET /api/market/dashboard
    GET /api/market/yield/personal
    ```
  </Tab>
</Tabs>

## Dashboard cards

<CardGroup cols={3}>
  <Card title="Portfolio Balance / Global Ecosystem Value">
    Displays the current total value in USD, formatted to two decimal places and updated every 100 ms via linear interpolation between Mesh pulses. In Sovereign Mode the label reads your name; in Collective Mode it reads "Global Ecosystem Value".
  </Card>
  <Card title="AI Strategy Status">
    Shows the current state of the neural execution layer. When active, this card displays `NEURAL_AUTO_ON` and reports the number of active neural orders running against your portfolio.
  </Card>
  <Card title="Guardian Link">
    Reflects the state of your Sovereign Guardian session. A verified session shows `SECURED` with the sub-label **Biometric ID Verified**. This card confirms that your mobile signature key is active and your session has not been revoked.
  </Card>
</CardGroup>

## Portfolio Performance chart

The chart spans two columns of the dashboard grid and renders a filled line graph of price or portfolio value over time. Use the time-filter buttons above the chart to change the visible window:

| Filter | Description |
|--------|-------------|
| `1D` | Last 24 hours (default) |
| `1W` | Last 7 days |
| `1M` | Last 30 days |
| `ALL` | Full available history |

In Collective Mode the chart dataset label is **Vexel Ecosystem Index**. In Sovereign Mode it is **Personal Portfolio performance**. Both datasets are sourced from their respective history endpoints on initialization and re-rendered whenever you switch the time filter.

<Note>
  The chart updates in real time as `marketPulse` socket events arrive from the Mesh. You do not need to interact with it to keep the data current.
</Note>

## AI Proactive Signals feed

The right column of the dashboard grid shows the **AI Proactive Signals** feed. This panel surfaces autonomous buy, sell, and strategy recommendations generated by the neural network, along with `CORE` and `SYSTEM` messages about Mesh activity and connection status.

Each actionable signal includes an **Approve** button. Clicking it opens the Sovereign Guardian modal, which requires a mobile signature before the trade is submitted.

<Warning>
  You must be in an active Sovereign Link session to approve trades. Unauthenticated users can view signals but cannot submit approvals.
</Warning>

See [AI Proactive Signals](/dashboard/ai-signals) for a full breakdown of signal types and how to act on them.

## Real-time updates via the Mesh

The dashboard subscribes to the `marketPulse` Socket.io event. The Mesh emits this event approximately every 15 seconds, delivering an updated `totalValue`, `yieldVelocity`, and `confidence` score.

When a pulse arrives:

- `currentNetWorth` and `yieldVelocity` are updated immediately.
- The `yieldVelocity` display refreshes to the new value formatted as `+$X.XXXX/sec`.
- The AI Confidence display updates to the new percentage.
- The system status indicator pulses green for 500 ms, then returns to blue.

Between pulses, a 100 ms interval interpolates the net worth forward, keeping the counter visually smooth without waiting for the next Mesh heartbeat.

For a detailed explanation of how Yield Velocity is calculated and displayed, see [What Is Yield Velocity?](/dashboard/yield-velocity).
